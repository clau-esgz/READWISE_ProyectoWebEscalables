ReadWise – Plataforma Web de Reseñas de Libros desarrollada con Angular

ReadWise es una aplicación web desarrollada como proyecto para la materia de Aplicaciones Web Escalables, utilizando Angular como framework principal para la construcción de interfaces dinámicas, junto con HTML, CSS y MongoDB para la gestión y almacenamiento de información.

La plataforma permite a los usuarios explorar un catálogo de libros, consultar detalles y acceder a reseñas realizadas por la comunidad. Los usuarios pueden organizar sus lecturas mediante diferentes categorías como libros por leer, libros leídos y libros reseñados, además de marcar su progreso y compartir opiniones sobre los libros que han terminado.

El sistema cuenta con diferentes tipos de usuario: usuario no registrado, usuario registrado y administrador. Los usuarios registrados pueden gestionar su biblioteca personal, mientras que el administrador tiene permisos para agregar nuevos libros, actualizar información y mantener el contenido de la plataforma.

Entre sus funcionalidades principales se incluyen la visualización detallada de libros, sistema de reseñas, buscador por criterios como ISBN y autor, gestión de usuarios y actualización dinámica de información mediante una aplicación escalable desarrollada con Angular.

Este proyecto permitió fortalecer conocimientos en desarrollo frontend, componentes de Angular, manejo de vistas, conexión con bases de datos, diseño de interfaces web y creación de aplicaciones con una estructura organizada y mantenible.
